# Plataforma Apollo

A **Plataforma Apollo** é uma plataforma digital de imóveis desenvolvida para facilitar a publicação, visualização e gestão de propriedades imobiliárias de forma moderna, rápida e segura.

---

##  Funcionalidades Principais

- Publicação de imóveis para venda e arrendamento  
- Visualização detalhada de cada imóvel  
- Gestão de usuários  
- Interface moderna e responsiva  

---

## Tecnologias Utilizadas

- Next.js  
- React  
- TypeScript  
- Node.js  
- Git & GitHub
